### Instructions:
Clone project
>git clone https://github.com/ArtursAsi/ToDoList

Install Composer
>composer install

Create .env file  
>cp .env.example.env

Link storage 
>php artisan Storage:link

Create Database 'todo_list_l' and migrate Database 
>php artisan migrate

Run app 
>php artisan serve


### Description

A simple ToDo list. Just add tasks.
